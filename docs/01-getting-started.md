# Start Coding in Your Language in 2 Minutes

1. Copy everything below  
2. Save as `my-first-program.html`  
3. Open the file → it works!

```html
<!DOCTYPE html>
<html>
<head><meta charset="utf-8"><title>My First Program</title></head>
<body>
  <h1>Sannu! Ẹ káàbọ̀! Ndewo!</h1>

  <!-- Change "hausa" → yoruba / igbo / pidgin if you want -->
  <script type="custom-js" data-language="hausa">
    rubuta("Sannu Duniya! Ina jin daɗi sosai")
    in (10 > 5) {
      nuna("Lissafi ya yi daidai!")
    }
  </script>

  <script src="../src/core/transpiler.js" type="module"></script>
</body>
</html>
```

Works on:

Any phone (Tecno, Itel, Infinix…)
Any laptop
Completely offline
No installation
