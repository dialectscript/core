# 05 Use Dialect Script in VS Code (2 minutes)

1. Install “Live Server” extension  
2. Open any .html file from this repo  
3. Right-click → “Open with Live Server”  
4. Edit → save → page auto-refreshes

Now you code in Hausa/Yorùbá/Igbo exactly like English devs.
