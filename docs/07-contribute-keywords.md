# 07 Add Keywords – No GitHub Needed!

You only need to know your language.

1. Go to → https://github.com/dialectscript/core/issues/new  
2. Choose template “Add keywords for my language”  
3. Write 5–10 lines like this:

Hausa: nuna = console.log
Yorùbá: fihan = console.log
Igbo: ziputa = console.log


That’s it. We add them in <24 h.

You just helped 100,000 kids. Thank you.
